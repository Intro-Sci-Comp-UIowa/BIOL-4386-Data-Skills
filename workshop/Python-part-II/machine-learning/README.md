## Data
The data used for this course is from <https://archive.ics.uci.edu/dataset/401/gene+expression+cancer+rna+seq>. To follow the workshop, either go to the above page, click "Download" button on the top right, unzip the downloaded file, and upload it to your IDAS home folder. Or, you can copy the file in the `classdata/machine-learning` folder. Then, use `tar xzvf TCGA-PANCAN-HiSeq-801x20531.tar.gz` at the terminal to extract the files, and then move both the `data.csv` and `labels.csv` from the expanded subfolder to the folder where the ipython notebook file is, and you are ready to carry out the workshop. Because the dataset is archived at the above website, the actual files are not tracked by the Git repository.
